**Функции, используемые в проекте:**

- Функция отображения всех товаров
- Функция отображения товара
- Функция добавления в корзину
- Функция отображения корзины
- Функция оформления заказа
---

**Модели данных:**

- Модель товара:
  - Название товара
  - Описание товара
  - Цена товара
  
- Модель корзины
  - Список товаров
